Looking for a effective and easy to use PHP MVC framework? This is it!

  • Open source

    Smll is a open source PHP framework built by Kristoffer "mbk" Olsson.

  • MVC

    Smll is built and modelled after other popular MVC frameworks such as .NET MVC.

  • Beginner friendly

    The framework is written with both aspiring and experianced webdevelopers in mind!
