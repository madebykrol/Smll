Example 1 - A Guestbook

Our first example is the old classic, a simple guestbook. Nothing fancy!

Step one.

Download and install Smll MVC.

Download the zip or tarball from the links in the header of this page and unpack it in a folder of choice.
Create a new project in your favorite IDE! Eclipse with PDT is recommended.

Step two

Create the database table.

CREATE TABLE `guestbook` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `name` varchar(50) DEFAULT NULL,
  `message` longtext,
  UNIQUE KEY `id` (`id`) 
);

Step three

The controlller

Open up and edit a file namned HomeController.php. Smll is setup by default to recognize this as the default controller, and requests to your site root will be routed to this. Edit the file to look something like this: p.s: We won't be using the About action in this example. 

namespace src\controllers;
class HomeController extends Controller {
  /**
   * @return ViewResult
   */
  public function index() {
    $this->viewBag['title'] = "Guestbook | My smll site";
    $model = new GuestbookFormModel();
    return $this->view($model);
  }
}

Step four

Views!

Go to and edit a the index.phtml file in your Views directory for the Home Controller views/Home/ to look something similar to this: 

<?php 
use smll\framework\helpers\Html;
$layout = "src/views/Share/_layout.phtml";
?>
<div class="row">
  <div class="guestbook form">
    <h2>Write in my guestbook</h2>
    <?php print Html::beginFormFor($model);?>
    
    <input type="submit" class="btn" value="submit" />
    <?php print Html::closeForm();?>
  </div>
</div>

Step 5.

Our first model.

Next up, we'll create the foundation for our model, for now just copy this code and we will cover how it works in another chapter. 

namespace src\models;
class GuestbookFormModel {
  
  /**
   * [FormField]
   * [Label=Name]
   * [InputType=text]
   * [Placeholder=Name]
   * @var string
   */
  public $name;
  
  /**
   * [FormField]
   * [Label=Message]
   * [InputType=textarea]
   * @var unknown
   */
  public $message;
  
}

Summary

Now, if you browse your new site you will see something like this.

Now we have created our first form using Smll MVC, right now our form does not do anything.
